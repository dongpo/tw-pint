== Conformance levels

[cols="1,3", options="header"]
|===
|Level |Description
|TW-PINT Core |Conformance to PINT shared invoice semantics and TW-PINT core rules.
|TW-PINT Domestic B2B |Domestic Taiwan B2B invoice exchange with Taiwan identifiers and Business Tax rules.
|TW-PINT Export |Export invoice exchange with foreign buyer handling, zero-rated treatment and foreign currency support.
|TW-PINT eGUI Bridge |Implementation guidance for mapping TW-PINT data to local eGUI-related processes.
|===
