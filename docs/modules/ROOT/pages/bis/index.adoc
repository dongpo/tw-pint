= Peppol International (PINT) model for Billing
:page-toclevels: 4

The TW-PINT Billing specification has been developed as Taiwan implementation guidance for the Peppol International (PINT) model for Billing. It provides a single consolidated documentation page while keeping each chapter in an independent AsciiDoc source file for maintenance.

[IMPORTANT]
====
This is a candidate specification prepared for review. It is not a final conformance source until formally approved and published through the applicable Peppol governance process.
====

== Introduction

This specification defines Taiwan-specific implementation guidance for PINT Billing. It uses Peppol as the international invoice exchange layer and positions Taiwan eGUI as the domestic tax reporting and compliance layer.

== Statement of status

This is a working draft for technical and policy review.

== Document structure

The documentation contains five consolidated chapters:

* Chapter 1 — Business processes
* Chapter 2 — Business information
* Chapter 3 — Tax information
* Chapter 4 — Rules
