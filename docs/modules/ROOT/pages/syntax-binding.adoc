= Syntax Binding

TW-PINT uses OASIS UBL 2.1 Invoice syntax.

The authoritative syntax binding sources are:

* `syntax-binding/TW-PINT-UBL-SyntaxBinding-v0.1.csv`
* `syntax-binding/TW-PINT-UBL-SyntaxBinding-v0.1.xlsx` when included

TW-PINT does not create a new XML syntax. Taiwan jurisdiction terms are mapped onto existing PINT/UBL structures.
