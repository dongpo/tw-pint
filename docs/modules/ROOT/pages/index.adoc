= TW-PINT Candidate Specification v1.0

Taiwan Jurisdiction Extension Candidate of the Peppol International Invoice Transaction (PINT).

[IMPORTANT]
====
This is a candidate specification prepared for community review. It is not an official OpenPeppol publication and has not been approved by an OpenPeppol authority or a Taiwan Peppol Authority.
====

== Purpose

TW-PINT defines a Taiwan jurisdictional specialization of PINT Billing for B2B invoice exchange. It uses Peppol as the international invoice exchange layer and positions Taiwan eGUI as the domestic tax reporting and compliance layer.

== Core proposition

TW-PINT is not a replacement of Taiwan eGUI. It is an interoperability layer that enables Taiwan businesses to exchange invoices through the Peppol network while preserving the ability to map invoice data to Taiwan tax reporting processes.
