Conformance

A conformant TW-PINT Invoice SHALL:

  1. conform to UBL 2.1 Invoice;

  2. conform to applicable PINT International shared rules;

  3. satisfy applicable TW-IBR rules;

  4. pass TW-PINT Schematron validation;

  5. use TW-PINT code lists where required.

This draft is not yet an official conformance source.
